# saller-init  [![Build Status](https://travis-ci.org/Wooleners/saller-init.svg?branch=master)](https://travis-ci.org/Wooleners/saller-init)

## Install

```bash
$ npm i saller-init -g --save
```

## Usage

```bash
$ mkdir saller-demo && cd saller-demo
$ saller-init
```

```bash
$ npm start
```
